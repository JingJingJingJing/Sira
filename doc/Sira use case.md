# 用例6
目标：交互式创建任务

角色：用户

步骤：

1. 输入创建命令
2. 程序提示必填字段
3. 用户输入相关信息
4. 重复步骤2，直至必填字段填写完毕
5. 程序输出创建结果

# 用例7
目标：交互式更新任务

角色：用户

步骤：

1. 输入更新命令
2. 程序输出所有备选字段
3. 用户
    1. 选择备选字段，输入备选字段
    2. 选择退出
4. 重复步骤2
5. 程序输出更新结果

# 用例9

3. 用户按上下键可以显示出命令A和命令B
